ADM Daggerheart (WIP) — Foundry VTT v13 System Scaffold
======================================================

Установка (локально)
1) Скачайте архив и распакуйте папку `adm-daggerheart` в:
   FoundryVTT/Data/systems/

2) Перезапустите Foundry и при создании мира выберите систему:
   "ADM Daggerheart (WIP)"

Что уже есть
- system.json + template.json
- Actor types: character, npc
- Item types: weapon, ability
- Простые листы (ActorSheetV2 / ItemSheetV2) + шаблоны Handlebars
- Локализация en/ru
